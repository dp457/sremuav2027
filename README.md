# SReMUAV 2027 Workshop Website

This is a GitHub Pages-ready website for:

**1st International Workshop on Security and Resilience of Multi-UAV Systems (SReMUAV 2027)**  
Co-located with IEEE PerCom 2027.

## Files

- `index.html` — main website content
- `styles.css` — website styling

## How to publish with GitHub Pages

1. Create a GitHub repository, for example: `2027` under the `sremuav` GitHub organization/user.
2. Upload `index.html` and `styles.css` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. The website should appear at:

```text
https://sremuav.github.io/2027/
```
